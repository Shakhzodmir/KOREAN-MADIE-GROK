# 🌸 Korean with Madie

**Корейский с первого шага — с любовью от Мади**

Красивое, нежное и вдохновляющее веб-приложение для изучения корейского языка. Для девушек, которые мечтают понимать дорамы без субтитров и петь BTS осмысленно.

## ✨ Особенности

- 🎀 **Красивое интро** с анимированным логотипом и падающими сакурами
- 📚 **30 уроков** с прогрессом и системой достижений
- 🧩 **Hangul Lab** — интерактивный конструктор слогов
- 📖 **Карточки слов** с системой интервального повторения (SM-2)
- 🎮 **5 мини-игр** для практики
- 📅 **Календарь корейских праздников** с ближайшими событиями
- 🔥 **Стрик и XP** — мотивационная система
- 🌸 **Полностью на русском + корейском**

## 🚀 Как запустить

### Локально
Просто откройте `index.html` в любом браузере.

### На GitHub Pages (рекомендуется)
1. Загрузите папку на GitHub
2. В настройках репозитория включите **GitHub Pages** (Source: Deploy from a branch → `main`)
3. Ваше приложение будет доступно по адресу:
   `https://ваш-юзернейм.github.io/korean-with-madie/`

## 📱 Дизайн

- Нежная розово-кремовая палитра
